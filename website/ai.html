Transformation of the data via an AI System


Feature extraction via Convolutional Neural Network

Once the data has been correcly found and explored, an AI system need to be setup in order to obtain some meaningful information from the larga amount of samples collected. In order to have each data point "understandable" by the machine, the audio files are first converted in a frequency-based reppresentation called the "input image" of the Convolutional Neural Network described next.

The frequency reppresentation of each sample contains all the informations we need about musical charachterestics helpful to determine a gernre (for example beat, tempo, etc.). To extract them, a Convolutional Neural Network (see image below) is used, allowing to extract and learn such useful features from the dataset itself. The output of such network is a new dataset containing 1000 rows (samples) each expressed with 1024 columns (features).

Dimensionality reduction via PCA and T-SNE

Having extracted 1024 features for each sample is a great achievement for the AI System which can now have all the informations needed to recognise each genre and automatically classify them. The System will identify each point in a new space of 1024 components (yes! not in 2D, nor in 3D but in crazy 1024D !) from which it will be possibile to identify some groupped songs: those are the genres.

Imagining a 1024D space makes our brain hurt, i know. This is why we need an extra step to plot such space in a more common one for us: a 3D space. To do so, the AI System will implement some techniques called PCA and T-SNE that will try to extrapolate the three most important features out of those 1024 obtained. Those will then be used to plot the space as coordinates. To better understand what the output of such algorithm looks like, you can download the resulting CSV file the computation by clicking here .
